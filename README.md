# TDD example project of Gilded Rose for node.js

## Teck Stack
- babel
- should
- mocha
- lodash
- gulp
- gulp-mocha
- code coverage: gulp-istanbul, isparta

## How to run

```javascripts
npm install //or yarn install
npm run test
npm run coverage
```
